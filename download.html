Cara membuat script html bucin versi ngajak jalan

Kali ini saya akan membagikan sebuah cara membuat Script HTML Bucin versi Ngajak Jalan Doi. Berikut ini tautan yang bisa langsung kamu lihat hasilnya:

Download Demo Share To WhatsApp

Langkah pertama yang harus dilakukan yaitu wajib memiliki aplikasi HTML Editor seperti Quick Editor (untuk Android) atau Notepad++ (untuk PC Windows). Selanjutnya, kalian wajib mengunduh Mentahan Script di bawah ini:
Mentahan.zip

Ekstrak File ZIP


Setelah semuanya sudah lengkap, buka mentahan file yang sudah diambil tadi dan jangan lupa di-ekstrak terlebih dahulu file .zip-nya agar tidak terjadi error. Ikuti langkahnya seperti di bawah ini.
setelah klik ekstrak lalu buka pake quick editor kalian.

Edit Kata-Kata


Nah, untuk mengedit kata-kata seperti gambar di bawah ini atau yang lainnya cukup gampang, kok.

   Keterangan:
1) Edit url didalam kode img src="" untuk mengganti gif/gambar.Kamu bisa mengupload fotonya ke situs postimages.org atau jika ingin mentahan gif nya ada di link ini https://fo-rpl.github.io/mentahan-gif.html
2) Untuk mengganti kata, edit pesan yang ada di antara kode span. 

Edit Kata akhir


Selanjutnya, untuk mengubah gambar, kata-kata akhir, peletakkan kodenya berbeda. Silakan cari beberapa pengkodean yang mirip seperti di bawah ini:
Nah.., setelah mengganti semua pesannya tinggal simpan deh, lalu tinggal kirim..
